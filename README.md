# BizTrack

BizTrack is a simple, no-code friendly invoice and expense tracking application for small businesses.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
